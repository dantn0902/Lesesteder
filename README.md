## App
Oversikt over leseplasser på UiO. Karakteristikker slik som kollokvierom, lys og støynivå legges til for hvert av stedene. 

Hva appen skal tilby:
- Åpningstider for betjent og ubetjent
- Beliggenhet (besøksadresse/kart) - ikke bruke mazemap
- Fikse "hos oss kan du" delen av uio nettsiden
- Kontakt informasjon (epost, telefon og postadresse)
- Antall sitteplasser
- Om det er kollokvierom
- Stillesone
- Kafe eller kantine i nærheten
- Bilde av stedet
- Antall printere og beliggenhet? (ifi har null printere i biblioteket)

Senere funksjonaliteter:
- Stikkontakter
- Lys
- Vindusplasser
- Temperatur
- Sti fra der du er
